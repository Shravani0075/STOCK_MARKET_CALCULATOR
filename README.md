# STOCK_MARKET_CALCULATOR
Stock Market Investment Calculator (C Programming)

This project is a simple C-based application designed to calculate stock market investment returns. It allows users to input investment amount, time duration, and expected rate of return to estimate profit and total value.

The program is built using core C programming concepts such as loops, functions, and user input handling, making it a beginner-friendly project for understanding financial calculations and structured programming.

🔧 Features
Calculates investment returns
User-friendly input system
Based on basic financial logic
Console-based application
🛠️ Tech Stack
Language: C
